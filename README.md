# PRODIGY_CS_02 - Image Encryption Tool

##  Description
This project is a GUI-based image encryption and decryption tool developed using Python. It allows users to securely transform images using pixel-level manipulation with a user-defined key.

## Features
- Encrypt images using a key-based pixel manipulation technique  
- Decrypt images using the same key  
- User-friendly GUI built with Tkinter  
- Browse and select images from system  
- Saves encrypted image in original folder  
- Displays output images in project directory  
- Clean and simple interface  

## 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- Pillow (PIL)
